Help text widget for WordPress

Display accordion help text in the WordPress widget areas. Includes jquery animation for opening and closing the help text when the title is clicked on.

Description

Display accordion help text in the WordPress widget areas. Includes jquery animation for opening and closing the help text when the title is clicked on.

Code contributions are welcome on Github.

Installation

1.Upload the dg-help-widget directory to the /wp-content/plugins/ directory 2. Activate the plugin through the 'Plugins' screen in your WordPress admin area 3. Upload your CSV file in the 'Users' menu, under 'Import From CSV'
